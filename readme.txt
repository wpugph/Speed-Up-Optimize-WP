== Description ==

Tool to help analyze scripts and css loaded in header and footer, useful in planning to do pagepseed optimizations.

== Installation ==

Installing "Speed Up Optimize" can be done either by searching for "Speed Up Optimize" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
